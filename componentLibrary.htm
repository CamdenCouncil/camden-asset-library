Component library for the LB of Camden residential account

A library of all the components currently used for the residential accounts web pages.

Responsive, mobile first approach, by default.

Components have been built to cater for large desktops ( greater than 1200px ), medium desktop ( greater than 992px ), tablets ( greater than 768px ) and mobile devices ( less than 768px ).

Components degrade gracefully to ie8. Where elements are not responsive in ie8, this is duly noted. The components are built using the bootstrap API ( current version 3.3.2 ).

Click on the buttons below to select the type of elements you need:

- Core components i.e. for the header, footer and left nav;
- Content components for the content area.

Code can be copy and pasted from the code box associated with each component or set of components.

Responsive Table

default table, default striped table, and a sortable table that is scrollable at a certain height and rows are odd/even coloured. Table widths are fluid by default, but can be a fixed width with a css override. Where you have multiple tables that need sorting, all tables will need unique id's assigned to them, and to initialise tablesorter. $("#tableA, #tableB, #tableC").tablesorter({ });. note: Table responsiveness and default striped table do not work in ie8. In these cases graceful degradation applies.
